## [0.1.0] - 2024-11-17

### Added
- Initial release of Paarvai MCP Server
- Natural language query support for AWS resources
- Relationship discovery between resources
- Security analysis tools
- IAM permission analysis
- Unused resource detection
- Data flow tracing
- Support for EC2, RDS, Lambda, S3, VPC, IAM, and more
- Comprehensive documentation and examples
